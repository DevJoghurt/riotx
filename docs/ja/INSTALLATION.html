インストール

インストール

ダウンロード, CDN

https://github.com/cam-inc/riotx/tree/develop/dist もしくは、下記の unpkg.com を利用してください。

重要 : unpkgは、無料のベストエフォートサービスです。稼働時間やサポートを保証していません。 https://unpkg.com

  • 非圧縮版
    • https://unpkg.com/riotx/dist/riotx.js
  • 圧縮版
    • https://unpkg.com/riotx/dist/riotx.min.js
  • バージョン指定
    • https://unpkg.com/riotx@1.0.0/dist/riotx.js

NPM

Webpack, Rollup はこちらを使用してください。

$ npm install --save riotx

https://www.npmjs.com/package/riotx

ソースからビルド

$ git clone https://github.com/cam-inc/riotx.git
$ npm install
$ npm run build
